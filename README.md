# London-Airbnb-Dataset-new
predicting Airbnb listing price
### Steps to Run
1. Clone the repository or download it as a ZIP file.
2. Open the notebooks/ directory.
3. Launch Jupyter Notebook or JupyterLab.
4. Open LDS7003M_ASSESSMENT.ipynb.
5. Import the necessary libraries;
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- xgboost
6. Run all cells from top to bottom to reproduce the analysis and results.
